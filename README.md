# Modularisation-assigment
In this example we have pizza types. Pepperoni, Veggie and Extra cheese. These classes all have a prepare_pizza method and a ready method which will be used when user requests a pizza from the factory module. We also have a pizza factory module which is the main working area. It has a Pizza factory class and inside it a static method called order_pizza(). Then the try block checks if the pizza type is valid or not or if the space is empty or not or at the very least there are any characters that are not alphabetical. After the input is entered the corresponding equality of the pizza type triggers an if statement. For different kinds of pizzas there are different outputs. Ending with a finally.
